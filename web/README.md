# MagicHours Web

Static HTML version of the MagicHours app. Host for free and add to your iPhone home screen.

## Host for Free

Upload the `index.html` file to any static host:

- **GitHub Pages**: Push to a repo → Settings → Pages → Deploy from `main` branch, root or `/web` folder
- **Netlify**: Drag `web` folder to [netlify.com/drop](https://app.netlify.com/drop)
- **Vercel**: `vercel web` or connect your repo
- **Cloudflare Pages**: Connect repo or upload folder

## Add to iPhone Home Screen

1. Open the site in **Safari**
2. Tap the **Share** button (square with arrow)
3. Scroll and tap **Add to Home Screen**
4. Name it (e.g. "MagicHours") and tap **Add**

The app will open full-screen without browser UI, like a native app.
